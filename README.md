# CivicBridge Kenya

**Bridging citizens to essential services**

CivicBridge Kenya is a static, information-focused website built as a frontend capstone project. It organizes common public service topics into clear, beginner-friendly guides (requirements, step-by-step processes, tips, and FAQs) so citizens can quickly understand what they need and what to do.

---
## Live Demo & Repository

If you want to preview the website use the [Live Demo](https://josephgakono.github.io/CivicBridge-Kenya/)

You can also access the repository : [Github Repository](https://github.com/josephgakono/CivicBridge-Kenya)

## Project Overview

Many Kenyans waste time moving between offices, cyber cafés, and conflicting online posts just to figure out basic processes like taxes, healthcare steps, student finance, or required documents. This project solves that by presenting structured, readable guides in one place.

**Key goals:**

- Provide clear, organized information in a simple interface
- Maintain consistent design across multiple pages
- Deliver a responsive layout that works on mobile and desktop
- Practice clean semantic HTML and modern CSS layout techniques

---

## Pages Included

The project includes **7+ pages** (capstone requirement):

- `index.html` – Homepage + navigation to all guides
- `passport.html` – IDs & Passport guide
- `kra.html` – KRA & Tax basics guide
- `healthcare.html` – Healthcare guide
- `helb.html` – HELB & student finance guide
- `jobs-cv.html` – Jobs & CV toolkit
- `business.html` – Business starter pack
- `contact.html` – Contact page (static demo form)

---

## Features

- **Black & white professional theme** (clean, readable, consistent)
- **Responsive navigation** including a **CSS-only mobile menu**
- **Reusable page layout pattern**:
  - Breadcrumbs
  - Sidebar “On this page” navigation
  - Requirements checklist
  - Step-by-step workflow
  - FAQ section using `<details>` and `<summary>`
- **Accessible, semantic HTML** structure (header, nav, main, section, footer)
- **No external libraries** (pure HTML + internal CSS)

---

## Technologies Used

- **HTML5**
- **CSS** (Flexbox + Grid)
- **Internal CSS** (per project requirement)

---

## How to Run

1. Download or clone the project folder.
2. Open `index.html` in a browser (Chrome/Edge/Firefox).
3. Use the navigation links to browse pages.

No installation, no build tools, and no backend required.

---

## Design Notes

This project uses a minimal, widely adopted UI approach:

- high contrast typography
- structured spacing system
- card-style content sections
- consistent navigation across all pages

The style aims to feel official and trustworthy while remaining modern and easy to scan.
